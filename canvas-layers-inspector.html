How It Works
Static — Grid + committed shapes. Cached; re-renders only on element mutation.
NewElement — Element being drawn. Isolated to avoid full scene re-renders during creation.
Interactive — Selection UI + handles. Only layer receiving pointer events. Re-renders on every mouse move.
